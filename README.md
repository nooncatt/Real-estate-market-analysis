# Анализ рынка недвижимости

Исследование факторов, влияющих на стоимость жилья, на основе реального датасета по объектам недвижимости.

## Цели проекта

Проект посвящён исследованию рынка недвижимости в России на основе реального датасета объявлений о продаже жилья.  

Основные цели:

1. Изучить структуру данных и выполнить первичный анализ (типы, пропуски, аномалии).
2. Объединить основной датасет с дополнительной информацией о регионах (`region.csv`).
3. Восстановить пропуски в `kitchen_area` логически обоснованным способом.
4. Исследовать рынок жилья в РФ за период 2018–2021 годов.  
   Основная метрика анализа — **стоимость за квадратный метр (`price_per_m2`)**.
5. Обработать аномалии и выбросы в данных для получения корректных оценок.
6. Определить сезонность и периоды, когда покупка или продажа наиболее выгодна.
7. Выявить факторы, которые влияют на стоимость недвижимости (количество комнат, площадь, этажность, регион и др.).
8. Сформулировать гипотезу о различиях стоимости жилья между группами объектов и проверить её статистическими методами.
9. Построить простую модель прогноза цены недвижимости и оценить её качество.

## Данные

- Формат: CSV / Excel
- Наблюдения: квартиры / дома с признаками (площадь, количество комнат, этаж, район и т.д.).
- Целевая переменная: стоимость объекта.

## Стек

- Python 3
- NumPy, Pandas
- Matplotlib, Seaborn
- Jupyter Notebook

## Что сделано

1. **Первичный анализ данных**
   - загружены данные, проверены типы, размеры выборки и наличие пропусков;
   - выявлены некорректные значения и аномалии в площади, кухонной зоне, этажности и стоимости.

2. **Объединение с региональной информацией**
   - загружен датасет `region.csv`;
   - выполнено аккуратное объединение таблиц по коду региона;
   - обработаны несовпадения и исправлены ошибки в кодировке регионов.

3. **Восстановление `kitchen_area`**
   - исследовано распределение кухонных площадей;
   - выбрана стратегия восстановления пропусков с учётом:
     - общей площади жилья,
     - количества комнат,
     - соотношения кухни к площади объекта;
   - пропуски заменены на обоснованные значения.

4. **Обработка аномалий**
   - удалены или скорректированы наблюдения с невозможными значениями (площадь < 10 м², кухня > общая площадь, нулевая цена и др.);
   - применена фильтрация выбросов для метрики `price_per_m2`.

5. **Исследовательский анализ рынка**
   - построены распределения ключевых признаков (площадь, комнаты, этажи, цена);
   - проанализирована динамика цен по годам;
   - выявлены сезонные изменения стоимости жилья;
   - определены периоды, когда цены были минимальны/максимальны.

6. **Анализ факторов, влияющих на стоимость**
   - рассчитана матрица корреляций количественных признаков;
   - построены графики зависимости стоимости от:
     - общей площади,
     - количества комнат,
     - региона,
     - типа здания,
     - этажности;
   - выделены признаки, оказывающие наибольшее влияние.

7. **Проверка гипотез**
   - сформулирована гипотеза о различиях стоимости жилья между группами объектов;
   - применены статистические критерии для проверки различий;
   - сделаны выводы на основе p-value и визуализаций.

8. **Модель прогноза цены**
   - создана выборка признаков;
   - обучена базовая модель линейной регрессии;
   - рассчитаны метрики качества модели (MSE, R²);
   - сформированы выводы о применимости модели к данному набору данных.

9. **Визуализации**
   - построены графики распределений, scatter-plots, boxplots и heatmap;
   - сохранены ключевые визуализации для отчёта.
