# Анализ рынка недвижимости

Исследование факторов, влияющих на стоимость жилья, на основе реального датасета по объектам недвижимости.

## Цели проекта

- изучить распределение цен на жильё;
- найти факторы, которые сильнее всего связаны с ценой (площадь, район, этаж, количество комнат и т.д.);
- построить визуализации и базовые статистические модели;
- сформулировать выводы, которые могут быть полезны для владельцев и покупателей.

## Данные

- Формат: CSV / Excel
- Наблюдения: квартиры / дома с признаками (площадь, количество комнат, этаж, район и т.д.).
- Целевая переменная: стоимость объекта.

## Стек

- Python 3
- NumPy, Pandas
- Matplotlib, Seaborn
- Jupyter Notebook

## Структура репозитория

- `kt_2_real_estate.ipynb` — основной ноутбук с анализом и визуализациями;
- `data/` — исходные данные;

## Что сделано

- проведена очистка данных: обработка пропусков, выбросов, приведение типов;
- исследовано распределение цен и ключевых признаков;
- рассчитаны корреляции между ценой и факторами;
- построены графики:
  - распределения цен,
  - зависимость цены от площади, количества комнат и района,
  - тепловая карта корреляций;
- сформулированы выводы о том, какие признаки сильнее всего влияют на стоимость жилья.
